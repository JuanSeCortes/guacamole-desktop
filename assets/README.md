# Assets - Iconos

Esta carpeta debe contener los iconos de la aplicación para diferentes plataformas:

- `icon.png` - Icono para Linux (512x512 o 1024x1024 píxeles)
- `icon.ico` - Icono para Windows (256x256 píxeles, formato ICO)
- `icon.icns` - Icono para macOS (1024x1024 píxeles, formato ICNS)

## Nota

Si no tienes iconos personalizados, la aplicación funcionará sin ellos usando los iconos predeterminados de Electron.

Para crear iconos puedes usar herramientas como:
- [Electron Icon Maker](https://www.electron.build/icons)
- [CloudConvert](https://cloudconvert.com/) para conversiones
- Herramientas de diseño como GIMP, Photoshop, etc.
